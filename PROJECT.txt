DressAI Super — bitta monorepo (web + API + mobil)
================================================================================

NIMA BO'LADI
- getdressai/     — veb-sayt (Supabase, til/region): katalogdagi mahsulotlarni
  o‘lcham/uslub/fason bo‘yicha moslik foizi, virtual kiyib ko‘rish (demo),
  hamkor havolalari (Amazon Associates tag). AI matn uchun POST /api/outfit —
  kalitlar serverda; Groq/OpenRouter (arzon) yoki
  Ollama (bepul lokal). Vercel yoki getdressai/SETUP.md; production VPS — deploy/hetzner/SETUP-HETZNER.txt (Render emas).
- dressai-api/    — Express: ro‘yxatdan o‘tish, JWT, Stripe PRO, /analyze
  (tavsiya + affiliate URL), /image (OpenAI rasmi). Mobil ilova shu yerga
  ulanadi.
- mobile/         — Expo (DressAI): foto, PRO, tavsiyalar va rasmlar.

FOYDALANUVCHI UCHUN QULAYLIK
- Veb: brauzerda ochiladi, bitta `npm run dev:web` (yoki VS Code vazifasi).
- Mobil: `cd mobile && npx expo start` — API manzili .env da EXPO_PUBLIC_API_URL.
  Telefonda Expo Go: kompyuter LAN IP va port 3000 (dressai-api ishlamoqda).

XARAJAT (qisqa)
- Veb matn-AI: Groq/OpenRouter tariflari + Vercel bepul limit; Ollama — 0$.
- Rasmlar va /analyze: OpenAI (qimmatroq) — faqat dressai-api da; PRO cheklovlari
  allaqachon serverda.
- Ma’lumotlar: veb uchun Supabase; API uchun hozircha data/users.json — production
  da disk yoki PostgreSQL/Supabase ga ko‘chiring (deploy yo‘qolmasin).

DAROMAD
- Stripe PRO (dressai-api + mobil IAP mavjud).
- AFFILIATE_TAG va Amazon tag vebda (getdressai) — ikkala tomonda affiliate.
- eBay Developer (OAuth): .env da EBAY_APP_ID, EBAY_CERT_ID, EBAY_DEV_ID, EBAY_RUNAME — getdressai/SETUP.md.

HYBRID VTON (GPU server, ixtiyoriy)
- FASHN / OmniTry / FastFit: getdressai/SETUP.md §7 (git clone + pip).
- Veb: index.html → GETDRESSAI_HYBRID_VTON_URL (.env) ёки /api/hybrid-vton; try-on аввал реал, кейин демо.
- dressai-api: HYBRID_VTON_SERVICE_URL → POST /v1/hybrid-vton (mobil Expo Try-on tab).

ISHGA TUSHIRISH
1) API:  copy dressai-api/.env.example → .env, kalitlarni to‘ldiring.
   cd dressai-api && npm install && npm run dev   → http://0.0.0.0:3000
2) Veb:  getdressai/.env — Supabase + AI kalitlari;  npm run dev:web  → 8787
3) Mobil: cd mobile && npm install && npx expo start
   Telefon: EXPO_PUBLIC_API_URL=http://<LAN-IP>:3000

VS Code: vazifalar — "GetdressAI: dev server" va "DressAI: API (3000)".

SIFAT / SUPER ROBOT
- Desktop\super_robot\run_dressai_quality.bat — getdressai npm test + dressai-api node --check
- Monorepo ildizida: npm run test:web
- CI: .github/workflows/ci.yml (GitHub’da)
